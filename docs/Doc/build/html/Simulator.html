Simulator.functions¶

Simulator.functions.smoothstep(x, y, radius, slope, x_center, y_center)¶

This function compute a 2D smooth symmetrique step function.

Args:
x (float): x value in the mesh. y (float): y value in the mesh radius (float): the radius of the step function. slope (float): slope of the erf function. x_center (float): the center in the x axis of the step function. y_center (float): the center in the y axis of the step function.
